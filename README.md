# school-project

<hr>

## Backend
### Из корня
- Запуск сервера с автоматическим обновлением при изменениях: <br>
``` uvicorn backend.app.main:app ``` <br>

<br>

- Запуск всех тестов: <br>
``` pytest -s backend/tests ``` <br>


## Frontend
### Из корня
- Запуск фронтэнда <br>
``` npm run dev --prefix frontend ``` <br>
